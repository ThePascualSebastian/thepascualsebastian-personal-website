01

Vibe-Hunter

My Role: Full-Stack Development & Testing

A distributed multiplayer Battleship game built as a REST API, where players join game lobbies, place ships, and fire at each other's boards through stateless HTTP endpoints. The system is engineered for scalability — a shared MySQL database persists all game state so any server instance can handle any request, with Postman-validated endpoints covering every game phase. Key architectural decision: all game logic lives in the API layer, keeping the client thin and the server the single source of truth for board state and turn management.

02

NBA Collection Manager

My Role: Solo Full-Stack Developer

A full-stack player collection manager with live CRUD operations, search, filtering by position, multi-column sorting, and a stats dashboard — all backed by a PHP REST API and MySQL database. The interesting challenge was building a single-page experience where every action (add, edit, delete, filter) hits the API and re-renders without a page reload, giving it the feel of a native app. Key architecture decision: all database logic is centralized in one PHP router using PDO with parameterized queries, making the API both secure and easy to extend with new endpoints.

My Specific Contributions

Sole developer — designed and built the complete frontend SPA, PHP REST API, MySQL schema, seed data pipeline, and deployment on InfinityFree with a custom domain and Let's Encrypt SSL.
